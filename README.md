# HolyShift

HolyShift is a Turtle WoW druid automation addon supporting Cat, Bear, and Balance (Moonkin) forms with single-target and AOE modes.

Maintainer: Miioon
Original author: Maulbatross (Kronos 3)

## Controls

/hsdps single
/hsdps aoe

These execute instantly.

## Forms

Cat:
- Rake → Shred/Claw → Rip → Ferocious Bite
- AOE: Swipe

Bear:
- Single: FF → Demo Roar → Maul queue
- AOE: Demo Roar → Swipe + Maul weave

Balance:
- Single: FF → IS → Moonfire → Starfire
- AOE: Moonfire spread → filler

## Notes
- Auto form detection
- Requires abilities on bars
